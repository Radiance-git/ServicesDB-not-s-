###  ساخت دیتابیس
قبل از هر کاری باید دیتابیس و جدول‌هاش ساخته بشن. فایل `q.sql` رو تو SQL Server Management Studio (یا هر ابزاری که باهاش وصل میشی) باز کن و اجراش کن. این فایل کل دیتابیس و ساختار جداول لازم رو می‌سازه، پس بدون اجرای این یکی، اسکریپت پایتون کار نمی‌کنه.(ترجیها دیتابیسی به اسم services نداشته باش)


## این اسکریپت و مخلفاتش چیکار میکنن؟

از فایل `data.csv` (نام سایت، URL، نام سازمان) میخونه، تو دیتابیس SQL Server یه سازمان و سرویس براش می‌سازه، بعد به هر URL یه درخواست می‌زنه ببینه بالاست یا نه و نتیجه (وضعیت + زمان پاسخ) رو ثبت می‌کنه.

⚠️ هر بار اجراش کنی، دوباره همه‌چی رو از اول insert می‌کنه (چک نمی‌کنه قبلاً بوده یا نه)، پس چندبار اجراش نکن وگرنه پر میشه از رکورد تکراری.



### ۱. پیش‌نیازها
پایتون باید رو سیستمت نصب باشه. یه چیز دیگه هم لازمه  **ODBC Driver for SQL Server**، چون پایتون بدون این نمی‌تونه به دیتابیس وصل شه.

* **نصب درایور دیتابیس (اجباریه، جا ننداز):** برو از سایت مایکروسافت **[ODBC Driver for SQL Server](https://learn.microsoft.com/en-us/sql/connect/odbc/download-odbc-driver-for-sql-server)** رو دانلود کن (نسخه ۱۷ یا ۱۸، فرقی نداره) و نصبش کن. اگه ویندوزت ۶۴ بیتیه (که احتمالاً هست)، نسخه x64 رو بگیر.

### ۲. نصب کتابخونه‌ها
اول ریپو رو دانلود کن. بعد تو پوشه اصلی پروژه یه ترمینال یا CMD باز کن و این رو بزن:
```bash
pip install -r requirements.txt
```

### ۳. اجرا کن
دیگه آماده‌ای. تو همون پوشه‌ای که `importdata.py` هست، این رو بزن (فقط حواست باشه نتت وصل باشه):
```bash
python importdata.py
```

### ۴. صبر پیشه کن
بذار کارش رو بکنه، بسته به سرعت نتت شاید ۵ تا ۱۰ دقیقه طول بکشه (به لطف زیرساخت ارتباطات قدرتمند کشور!)
